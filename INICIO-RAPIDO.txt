🚀 INÍCIO RÁPIDO - Site Aliado
================================

Para visualizar o site agora mesmo:

1️⃣ Opção Simples (Navegador)
   - Abra a pasta 'aliado'
   - Clique duas vezes em 'index.html'
   - Pronto! O site abrirá no navegador

2️⃣ Opção com Servidor Local (Recomendado)

   No PowerShell, dentro da pasta 'aliado':
   
   python -m http.server 8000
   
   Depois acesse: http://localhost:8000

3️⃣ Opção VS Code Live Server
   - Instale a extensão "Live Server"
   - Clique com botão direito em 'index.html'
   - Selecione "Open with Live Server"

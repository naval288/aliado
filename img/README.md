# Imagens do Site Aliado

Esta pasta contém as imagens utilizadas no site.

## Estrutura de Pastas

```
img/
├── carousel/          # Imagens do carrossel principal
│   ├── 1.jpg         # Primeira imagem (1920x600px recomendado)
│   ├── 2.jpg         # Segunda imagem (1920x600px recomendado)
│   └── 3.jpg         # Terceira imagem (1920x600px recomendado)
└── sobre.jpg         # Imagem da seção "Sobre" (800x600px recomendado)
```

## Como Adicionar Suas Imagens

1. **Imagens do Carrossel**: Adicione 3 imagens na pasta `carousel/` com os nomes `1.jpg`, `2.jpg` e `3.jpg`
   - Resolução recomendada: 1920x600 pixels
   - Formato: JPG ou PNG
   - Imagens de alta qualidade relacionadas a educação/estudos

2. **Imagem Sobre Nós**: Adicione uma imagem chamada `sobre.jpg` nesta pasta
   - Resolução recomendada: 800x600 pixels
   - Formato: JPG ou PNG
   - Pode ser uma foto da equipe, estudantes, ou ambiente de estudo

## Fontes de Imagens Gratuitas

Você pode baixar imagens gratuitas e de qualidade nesses sites:

- [Unsplash](https://unsplash.com/) - Busque por: "education", "study", "learning", "students"
- [Pexels](https://pexels.com/) - Imagens gratuitas de alta qualidade
- [Pixabay](https://pixabay.com/) - Banco de imagens livre de direitos autorais

## Placeholders Temporários

Enquanto você não adiciona suas próprias imagens, você pode usar URLs de placeholder:

```html
<!-- Exemplo de uso -->
<img src="https://via.placeholder.com/1920x600/0d6efd/ffffff?text=Aliado" alt="Placeholder">
```

Ou usar o serviço Lorem Picsum para imagens aleatórias:

```html
<img src="https://picsum.photos/1920/600" alt="Random Image">
```

## Dicas

- Use imagens com boa iluminação e foco nítido
- Evite imagens muito escuras (o site já aplica um filtro de escurecimento no carrossel)
- Certifique-se de ter os direitos de uso das imagens
- Otimize as imagens antes de fazer upload (use ferramentas como TinyPNG)
- Mantenha o tamanho dos arquivos abaixo de 500KB para melhor performance
